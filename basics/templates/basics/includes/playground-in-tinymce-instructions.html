Instructions for adding JS Console, Gists and CodePens:

  1. Choose 'Tools' from the top menu, then select 'Source Code'.

  2. Select where in the article you wish to add the elements.

  3. Wrap each element in a <p></p> tag.

  4. When finished, remember to save!
